# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/SAHHITH/portfolio.git
```

Move into the project

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
